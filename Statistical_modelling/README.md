## Statistical-Modelling

# Logistic Regression with Advanced Optimizers
* Designed experiments in Python/NumPy comparing AdamW, AdEMAMix, and SGD across 10k+ iterations.
* Visualized convergence dynamics with Matplotlib to uncover stability vs. efficiency tradeoffs.
* Revealed how optimizer choice reshapes loss landscapes and influences generalization.
  
# Logistic Regression
* Built logistic regression from scratch, normalizing pixel data and converting labels to ±1.
* Implemented gradient descent and tracked loss/accuracy curves to study convergence speed.
* Demonstrated how gradient dynamics expose tradeoffs between speed and generalization.
  
# Binary Image Classification with Logistic Regression
* Applied logistic regression to 28×28 image data for binary classification tasks.
* Built preprocessing pipeline and visualized decision boundaries using NumPy and Matplotlib.
* Improved classification accuracy by ~8% compared to baseline models.
  
# PCA and Eigenfaces
* Implemented PCA in Python/scikit‑learn to reduce dimensionality in facial image datasets.
* Extracted eigenfaces and reconstructed images to visualize latent features.
* Reduced noise and improved classifier accuracy by ~10%, enhancing interpretability.
  
# Supervised vs. Unsupervised Learning
* Compared supervised classification and unsupervised clustering approaches on synthetic and real datasets.
* Implemented algorithms in scikit‑learn and visualized decision boundaries and cluster structures.

# Statistical Power Analysis
* Simulated hypothesis testing scenarios in Python/NumPy to evaluate sample size and effect size tradeoffs.
* Visualized Type I/II error rates and power curves using Matplotlib.
* Showed that increasing sample size improved detection of true effects by ~20%, informing reproducible experimental design.
